# Airline names

Look up airline names from their carrier codes.

## Dataset

CSV file: [nyc_airlines.csv](../data/nyc_airlines.csv)

Table name: `nyc_airlines`

## Format

<dl>
  <dt>carrier</dt>
  <dd>Two letter abbreviation.</dd>
  <dt>name</dt>
  <dd>Full name.</dd>
</dl>
